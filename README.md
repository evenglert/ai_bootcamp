# ai_bootcamp
Python labs and use cases in AI bootcamp:
* List of Python packages required for the AI bootcamp scripts
* evenglert_io_homework: Managing a student data base (add, remove, view, search for students)
* evenglert_data_exploration_iris_dataset: Data Exploration using Pandas to calculate various statistical measures for iris dataset 
* evenglert_supervised_machine_learning: Develop a machine learning model that can perform a data analysis on synthetically generated data; using data preprocessing, hyperparametr tuning, cross-validation; comparing performance of Linear Regression, Regression Tree, Random Forest, XGBoost, and Support Vector Machine
